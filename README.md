# 🖼️ PictoQuery

**PictoQuery** is an AI-powered image analysis application that helps users understand and interpret images through natural language conversations.

Built with **Streamlit** and powered by **OpenAI's Vision capabilities**, the application analyzes uploaded images and provides contextual explanations, helping users gain insights from visual content without requiring technical expertise.

## ✨ Features

* Upload and analyze images
* AI-powered visual understanding
* Ask follow-up questions about an image
* Context-aware responses
* Simple and intuitive Streamlit interface

## 🛠️ Tech Stack

* Python
* Streamlit
* OpenAI API

## Why I Built This

I wanted to explore how multimodal AI models interpret visual information and how natural language interfaces can make image understanding more accessible.

## Future Improvements
